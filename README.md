# BruControl Releases

## About this repo

This repository is used for hosting BruControl's plugin registry, firmware index, and version management. It serves as the central registry that BruControl applications use to discover and install plugins, and to fetch firmware updates.

**BruControl is not open source software** and this repo _does not_ contain the source code of BruControl. This repository only hosts:

- **Plugin registry** — manifests and version metadata for plugins available in BruControl
- **Firmware index** — version information and download references for supported hardware
- **Version management** — structured data used by BruControl for updates and compatibility

If you wish to extend BruControl, you can do so through the plugin system. The default plugins and their source code live in the [plugin-library](https://github.com/brucontrol/plugin-library) repository, which is open source.

## Structure

- `plugins.json` — Generated index of all registered plugins (do not edit manually)
- `plugins/` — Individual plugin manifests (`{id}.json`), where `id` is a UUID v5
- `firmware.json` — Firmware versions and file references

## Contributing

Plugin submissions and registry updates are managed through the plugin-library repository. Do not edit `plugins.json` manually—it is generated by the GitHub Action.

For questions about BruControl or this registry, please refer to the BruControl community or documentation.
